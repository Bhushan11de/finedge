A comprehensive stock market prediction web application based on **Machine Learning** and **Sentiment Analysis** of financial news and tweets.

## Features

### For Users:
- Real-time stock price tracking
- 7-day price predictions for NASDAQ and NSE stocks
- Financial news aggregation
- Currency converter
- Stock market education resources
- Downloadable stock tickers list

### For Admins:
- User management (CRUD operations)
- Email notification system
- All user features plus administrative controls

## Technology Stack

**Backend:**
- Python
- Django
- Machine Learning (TensorFlow/Keras)
- Pandas/Numpy for data analysis

**Frontend:**
- React.js
- Bootstrap
- HTML5/CSS3
- JavaScript/jQuery

**Data Sources:**
- Yahoo Finance API
- Twitter API for sentiment analysis
- Financial news APIs
